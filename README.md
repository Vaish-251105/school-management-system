# school-management-system
Internship Project - 120 hrs

# Project Overview
The School Management System is a complete digital ERP solution designed to manage and 
automate all academic and administrative operations of a school.  
This system includes a web-based Admin CRM and mobile applications for Teachers, 
Students, and Parents.  
The backend is developed using Node.js and MongoDB, and the mobile application is built 
using Flutter. 
The system improves efficiency, reduces manual paperwork, ensures real-time data access, 
and enhances communication between school, teachers, students, and parents. 

## Features - Project Objectives
- Automate school administrative operations 
- Digital student admission and records management
- Teacher and staff management
- Attendance tracking for students and teachers
- Online fee collection and receipt generation
- Academic management including exams and results
- Real-time notifications and communication
- Mobile access for parents, students, and teache
