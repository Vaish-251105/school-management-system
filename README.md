# 🎓 School Management System (ERP)

A fully dynamic and responsive **School Management System** built with modern technologies, supporting multiple user roles and complete CRUD operations across all modules.

---

## 🚀 Project Overview

This project is a **full-stack School ERP system** with both:

* 🌐 Web Application
* 📱 Flutter Mobile Application

It allows seamless management of students, staff, academics, communication, and finances with **role-based access control**.

---

## 👥 User Roles

The system supports the following roles:

* 🛠️ Admin
* 👨‍🏫 Teacher
* 🎓 Student
* 👨‍👩‍👧 Parent
* 💰 Accountant

Each role has **custom dashboards and permissions**.

### 🔐 Authentication & Authorization

* Secure login system
* Role-Based Access Control (RBAC)
* Session handling & logout functionality

---

### 🛠️ Admin Module

* Manage students, teachers, and staff
* Staff attendance management
* View reports and system overview

---

### 👨‍🏫 Teacher Module

* Mark and manage student attendance
* Manage timetable (view, update, download PDF)
* Create and manage exams & results
* Assign homework and track submissions
* View student fee status (paid/pending)

---

### 🎓 Student Module

* View attendance (read-only)
* Access report cards (download as PDF)
* View assignments and academic updates
* Communicate with teachers

---

### 👨‍👩‍👧 Parent Module

* Pay fees online
* View transaction history
* Download fee receipts (PDF)
* Access child’s attendance & results
* View important notices and holidays

---

### 💰 Accountant Module

* Manage student fees and transactions
* Manage staff and teacher salaries
* View financial reports and revenue summary
* Track attendance of staff and teachers

---

### 💬 Communication System

* Chat between students and teachers
* Internal messaging system

---

### 📅 Academic Features

* Dynamic calendar with important dates
* Exam scheduling and result management

---

### 🚌 Transport Module

* Bus tracking system
* Map-based route monitoring

## 📦 Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/school-management-system.git
cd school-management-system
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm start
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

### 4️⃣ Mobile App Setup (Flutter)

```bash
cd mobile
flutter pub get
flutter run
```

---

## 📱 APK Build (Android)

To generate APK:

```bash
flutter build apk --release
```

Output file:

```
build/app/outputs/flutter-apk/app-release.apk
```
